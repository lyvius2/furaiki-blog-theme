# 네이버 블로그 스타일 Blogger 템플릿

네이버 블로그의 깔끔한 디자인과 UX를 Blogger 플랫폼에서 사용할 수 있도록 제작한 HTML 템플릿입니다.

## 주요 특징

✨ **네이버 블로그 스타일 디자인**
- 깔끔하고 현대적인 인터페이스
- 카드 스타일의 포스트 레이아웃
- **대표 이미지 썸네일** 자동 표시 ⭐ 신기능!
- 직관적인 네비게이션

🎨 **세련된 UI/UX**
- 그라디언트 헤더 디자인
- 부드러운 호버 효과와 애니메이션
- 시각적으로 구분된 콘텐츠 영역

📱 **완벽한 반응형 디자인**
- 모바일, 태블릿, 데스크톱 모두 지원
- 유연한 레이아웃 구조
- 4단계 브레이크포인트 (1024px, 768px, 480px, 360px)
- 모든 화면 크기에서 최적화된 UX

🔧 **풍부한 위젯 지원**
- 프로필 위젯
- **계층적 카테고리 (라벨) 위젯** ⭐ 신기능!
  - 첫 번째 label = 루트 카테고리
  - 두 번째 label = 서브 카테고리
  - 접기/펼치기 기능
- 최근 글 목록
- 검색 기능
- 댓글 시스템

## 설치 방법

### 1. Blogger 대시보드 접속
1. [Blogger](https://www.blogger.com)에 로그인합니다.
2. 템플릿을 적용할 블로그를 선택합니다.

### 2. 템플릿 백업 (중요!)
1. 왼쪽 메뉴에서 **테마(Theme)** 를 클릭합니다.
2. 우측 상단의 점 3개 메뉴를 클릭하고 **백업 / 복원**을 선택합니다.
3. **테마 다운로드**를 클릭하여 현재 템플릿을 백업합니다.

### 3. 새 템플릿 업로드
1. **백업 / 복원** 화면에서 **업로드** 버튼을 클릭합니다.
2. 다운로드 받은 `blogger-template.xml` 파일을 선택합니다.
3. 업로드가 완료되면 자동으로 새 템플릿이 적용됩니다.

## 사용자 정의 가이드

### 헤더 색상 변경
템플릿의 CSS 부분에서 헤더 그라디언트 색상을 변경할 수 있습니다:

```css
#header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

원하는 색상 코드로 `#667eea`와 `#764ba2`를 변경하세요.

### 강조 색상 (포인트 컬러) 변경
네이버 블로그의 녹색 강조 색상을 다른 색으로 변경하려면:

```css
a {
    color: #03c75a;  /* 링크 색상 */
}

a:hover {
    color: #00a844;  /* 호버 시 색상 */
}
```

### 레이아웃 너비 조정
전체 레이아웃의 최대 너비를 조정하려면:

```css
#wrapper {
    max-width: 1200px;  /* 원하는 너비로 변경 */
}
```

## 위젯 설정

### 프로필 위젯 설정
1. Blogger 대시보드 → **레이아웃**
2. 사이드바의 **프로필** 위젯을 찾아 편집 아이콘 클릭
3. 프로필 사진 및 자기소개 내용 입력

### 포스트 썸네일 이미지

이 템플릿은 **포스트의 대표 이미지를 자동으로 썸네일로 표시**합니다!

#### 작동 방식
- 포스트 본문의 **첫 번째 이미지**가 자동으로 썸네일로 사용됩니다
- 이미지가 없는 포스트는 썸네일 영역이 표시되지 않습니다
- 썸네일을 클릭하면 해당 포스트로 이동합니다

#### 썸네일 특징
- **크기**: 가로 전체 × 세로 250px (데스크톱)
- **반응형**: 태블릿 200px, 모바일 180px로 자동 조정
- **효과**: 호버 시 1.05배 확대 애니메이션
- **자르기**: `object-fit: cover`로 비율 유지하며 영역에 맞춤

#### 썸네일 이미지 권장사항
- **권장 비율**: 16:9 또는 2:1
- **권장 크기**: 최소 800px × 400px
- **파일 형식**: JPG, PNG, WebP
- **최적화**: 웹용으로 최적화된 이미지 사용 (100-300KB 권장)

#### 이미지 없는 포스트
- 이미지가 없으면 썸네일 영역이 자동으로 숨겨집니다
- 제목과 본문만 표시되는 깔끔한 레이아웃으로 전환

### 카테고리 설정 (계층적 구조)

이 템플릿은 **계층적 카테고리 구조**를 지원합니다!

#### 작동 방식
1. 포스트에 2개 이상의 **라벨(Label)** 추가
2. **첫 번째 라벨** = 루트 카테고리 (상위 카테고리)
3. **두 번째 라벨** = 서브 카테고리 (하위 카테고리)

#### 예시
```
포스트 1: 라벨 = ["기술", "프로그래밍"]
  → 루트: 기술, 서브: 프로그래밍

포스트 2: 라벨 = ["기술", "웹개발"]
  → 루트: 기술, 서브: 웹개발

포스트 3: 라벨 = ["일상", "여행"]
  → 루트: 일상, 서브: 여행
```

#### 결과
사이드바에 다음과 같이 표시됩니다:
```
▼ 기술 (2)
  └ 프로그래밍 (1)
  └ 웹개발 (1)
▼ 일상 (1)
  └ 여행 (1)
```

#### 접기/펼치기 기능
- **▼ 아이콘**을 클릭하면 서브 카테고리를 접거나 펼칠 수 있습니다
- 초기 상태는 모두 펼쳐진 상태입니다

#### 주의사항
- 라벨이 1개만 있는 포스트는 계층 구조에 포함되지 않습니다
- 3개 이상의 라벨이 있어도 첫 번째와 두 번째만 사용됩니다

### 메뉴 커스터마이징
네비게이션 메뉴를 수정하려면 템플릿의 다음 부분을 편집하세요:

```html
<div id='nav'>
    <ul>
        <li><a expr:href='data:blog.homepageUrl'>홈</a></li>
        <li><a href='#'>카테고리</a></li>
        <li><a href='#'>방명록</a></li>
    </ul>
</div>
```

## 템플릿 구조

```
blogger-template.xml
├── 헤더 (Header)
│   ├── 블로그 제목
│   └── 설명
├── 네비게이션 (Navigation)
├── 메인 콘텐츠 영역
│   ├── 포스트 목록
│   ├── 포스트 상세
│   └── 댓글
├── 사이드바
│   ├── 프로필 위젯
│   ├── 검색 위젯
│   ├── 카테고리 위젯 (계층적 구조) ⭐
│   └── 최근 글 위젯
├── 푸터 (Footer)
└── JavaScript
    └── 계층적 카테고리 생성 스크립트
```

### 계층적 카테고리 구현 방식

이 템플릿은 **JavaScript**를 사용하여 Blogger의 JSON Feed를 분석하고 동적으로 계층적 카테고리 구조를 생성합니다.

**작동 원리:**
1. 페이지 로딩 시 Blogger의 `/feeds/posts/default?alt=json` API 호출
2. 모든 포스트의 label 데이터 수집
3. 각 포스트의 첫 번째 label을 루트로, 두 번째 label을 서브로 분류
4. 카테고리별 게시물 수 자동 계산
5. 계층적 HTML 구조 동적 생성
6. 접기/펼치기 이벤트 리스너 추가

## 스타일 특징

### 색상 팔레트
- **주 배경색**: `#f5f6f7` (밝은 회색)
- **카드 배경**: `#ffffff` (흰색)
- **강조 색상**: `#03c75a` (네이버 그린)
- **텍스트**: `#333333` (진한 회색)
- **보조 텍스트**: `#8b95a1` (중간 회색)
- **테두리**: `#e4e8eb` (연한 회색)

### 타이포그래피
- **기본 폰트**: 시스템 폰트 (맑은 고딕, Segoe UI 등)
- **기본 크기**: 14px
- **제목 크기**: 24px (포스트), 32px (헤더)

### 간격 시스템
- **카드 여백**: 30px
- **요소 간 간격**: 20px
- **작은 간격**: 10px

## 반응형 웹 브레이크포인트

템플릿은 다음과 같은 화면 크기에 최적화되어 있습니다:

### 🖥️ 데스크톱 (1025px 이상)
- 2단 레이아웃 (메인 콘텐츠 + 사이드바)
- 최대 너비 1200px
- 넓은 여백과 간격

### 💻 태블릿 가로 (769px ~ 1024px)
- 2단 레이아웃 유지
- 좁아진 여백
- 그림자 효과 제거로 더 넓은 화면 활용

### 📱 태블릿 세로 (481px ~ 768px)
- 1단 레이아웃으로 전환
- 사이드바가 콘텐츠 하단으로 이동
- 폰트 크기 및 여백 조정
- 네비게이션 메뉴 자동 줄바꿈

### 📱 모바일 (481px 이하)
- 1단 레이아웃
- 더 작은 폰트와 여백
- 터치 친화적인 버튼 크기
- 최소 여백으로 최대 콘텐츠 표시

### 📱 작은 모바일 (360px 이하)
- 초소형 화면 최적화
- 필수 정보 위주 표시
- 더욱 압축된 레이아웃

## 문제 해결

### 템플릿이 적용되지 않을 때
1. XML 파일이 올바르게 업로드 되었는지 확인
2. 브라우저 캐시를 삭제하고 새로고침 (Ctrl + F5)
3. 백업한 템플릿으로 복원 후 다시 시도

### 위젯이 표시되지 않을 때
1. Blogger 대시보드 → **레이아웃** 메뉴 확인
2. 해당 위젯이 활성화되어 있는지 확인
3. 위젯 설정에서 필요한 정보가 입력되어 있는지 확인

### 모바일에서 레이아웃이 깨질 때
1. Blogger 대시보드 → **설정** → **기타**
2. **모바일 템플릿 사용 여부** 확인
3. "데스크톱 버전 표시" 또는 "커스텀 모바일 템플릿" 선택
4. 브라우저에서 강제 새로고침 (Ctrl + Shift + R)

### 반응형이 작동하지 않을 때
1. 템플릿 파일에서 viewport 메타 태그 확인:
   ```html
   <meta content='width=device-width,initial-scale=1.0,minimum-scale=1.0,maximum-scale=5.0' name='viewport'/>
   ```
2. Blogger 설정에서 모바일 테마가 분리되어 있지 않은지 확인
3. 브라우저의 개발자 도구 (F12)로 미디어 쿼리 작동 여부 확인

### 계층적 카테고리가 표시되지 않을 때
1. **포스트에 라벨이 2개 이상** 있는지 확인
   - 라벨이 1개만 있으면 계층 구조에 포함되지 않습니다
2. 브라우저 콘솔(F12)에서 JavaScript 오류 확인
3. Blogger 블로그가 **비공개 상태**인 경우 JSON Feed 접근이 제한될 수 있습니다
   - 블로그를 공개로 설정하거나
   - 템플릿의 fetch URL을 확인하세요
4. 캐시 문제일 수 있으니 강제 새로고침 (Ctrl + Shift + R)

### 카테고리가 평면적으로 표시될 때
- JavaScript가 제대로 로딩되지 않았을 가능성
- 브라우저 콘솔에서 에러 메시지 확인
- `/feeds/posts/default?alt=json` URL이 정상 작동하는지 확인
  - 브라우저에서 `https://your-blog.blogspot.com/feeds/posts/default?alt=json` 직접 접속해보기
