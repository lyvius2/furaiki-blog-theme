# 미니멀 스타일 Blogger 템플릿

깔끔하고 미니멀한 디자인의 Blogger 템플릿입니다. 나눔명조 폰트와 세련된 레이아웃으로 가독성 높은 블로그를 만들 수 있습니다.

## 주요 특징

### 🎨 미니멀한 디자인
- 왼쪽 검은색 세로선이 있는 깔끔한 헤더
- 흰색 배경의 심플한 레이아웃
- 나눔명조 폰트로 세련된 타이포그래피
- 숫자는 Georgia 폰트로 표시

### 📝 포스트 기능
- **자동 썸네일**: 포스트의 첫 번째 이미지가 자동으로 썸네일로 표시 (목록 페이지에서만)
- **전체 본문 표시**: 단일 포스트 페이지에서는 전체 본문이 표시됩니다
- **중앙 정렬 이미지**: 포스트 본문의 이미지가 자동으로 중앙 정렬됩니다

### 🗂️ 계층적 카테고리 시스템
- **">" 구분자 사용**: 라벨을 "일본>2024 홋카이도" 형식으로 입력하면 자동으로 계층 구조 생성
- **자동 분류**: 첫 번째 부분이 루트 카테고리, 나머지가 서브 카테고리
- **접기/펼치기**: 서브 카테고리를 접고 펼칠 수 있습니다
- **">" 구분자가 없는 라벨은 무시**: 계층 구조에 포함되지 않습니다

#### 사용 예시
```
라벨: "일본>2024 홋카이도"
→ 루트: 일본
→ 서브: 2024 홋카이도

라벨: "일본>도쿄>시부야"
→ 루트: 일본
→ 서브: 도쿄>시부야
```

### 💬 댓글 시스템
- 프로필 이미지와 이름이 돋보이는 디자인
- 작성 시간이 오른쪽에 작은 회색 텍스트로 표시
- 나눔명조 폰트로 통일된 스타일
- 깔끔한 레이아웃과 간격

### 👤 프로필 위젯
- 사각형 프로필 이미지
- 이미지 오른쪽에 이름 표시
- 컴팩트한 레이아웃

### 🔍 검색 기능
- 이모지 아이콘 버튼 (🔎)
- 연한 회색 배경의 미니멀한 디자인

### 📱 완벽한 반응형 디자인
- 모바일, 태블릿, 데스크톱 모두 지원
- 4단계 브레이크포인트 (1024px, 768px, 480px, 360px)
- 모든 화면 크기에서 최적화된 UX

## 설치 방법

### 1. Blogger 대시보드 접속
1. [Blogger](https://www.blogger.com)에 로그인합니다.
2. 템플릿을 적용할 블로그를 선택합니다.

### 2. 템플릿 백업 (중요!)
1. 왼쪽 메뉴에서 **테마(Theme)** 를 클릭합니다.
2. 우측 상단의 점 3개 메뉴를 클릭하고 **백업 / 복원**을 선택합니다.
3. **테마 다운로드**를 클릭하여 현재 템플릿을 백업합니다.

### 3. 새 템플릿 업로드
1. **백업 / 복원** 화면에서 **업로드** 버튼을 클릭합니다.
2. 다운로드 받은 `blogger-template.xml` 파일을 선택합니다.
3. 업로드가 완료되면 자동으로 새 템플릿이 적용됩니다.

## 사용 가이드

### 계층적 카테고리 설정

#### 1. 라벨 형식
포스트에 라벨을 추가할 때 ">" 구분자를 사용합니다:

```
예시:
- "일본>2024 홋카이도"
- "일본>도쿄>시부야"
- "음식>일식>라멘"
```

#### 2. 작동 방식
- **">" 구분자가 있는 라벨만** 계층 구조에 포함됩니다
- 구분자가 없는 라벨은 카테고리에 표시되지 않습니다
- 첫 번째 ">" 앞의 텍스트가 루트 카테고리가 됩니다
- ">" 뒤의 모든 텍스트가 서브 카테고리가 됩니다

#### 3. 결과 예시
```
라벨: "일본>2024 홋카이도" (포스트 1개)
라벨: "일본>도쿄" (포스트 2개)
라벨: "일본>오사카" (포스트 1개)

사이드바 표시:
▼ 일본 (4)
  └ 2024 홋카이도 (1)
  └ 도쿄 (2)
  └ 오사카 (1)
```

### 프로필 위젯 설정
1. Blogger 대시보드 → **레이아웃**
2. 사이드바의 **프로필** 위젯을 찾아 편집 아이콘 클릭
3. 프로필 사진 및 자기소개 내용 입력
4. 프로필 이미지는 사각형으로 표시되고, 이름은 이미지 오른쪽에 표시됩니다

### 댓글 설정
1. Blogger 대시보드 → **설정** → **게시물, 댓글 및 공유**
2. **댓글** 섹션에서 댓글 허용 여부 확인
3. 댓글 위치를 "포스트 아래"로 설정
4. 댓글 모드 선택 (Google 계정, 익명 등)

## 커스터마이징

### 폰트 변경
기본 폰트는 나눔명조입니다. 다른 폰트로 변경하려면:

```css
body {
    font-family: '원하는 폰트', serif;
}
```

### 색상 변경
주요 색상은 CSS에서 변경할 수 있습니다:

```css
/* 강조 색상 */
a {
    color: #03c75a;
}

/* 배경색 */
body {
    background-color: #f5f6f7;
}
```

### 레이아웃 너비 조정
```css
#wrapper {
    max-width: 1200px;  /* 원하는 너비로 변경 */
}
```

## 템플릿 구조

```
blogger-template.xml
├── 헤더 (Header)
│   ├── 왼쪽 검은색 세로선
│   ├── 네비게이션 메뉴
│   └── 블로그 제목
├── 메인 콘텐츠 영역
│   ├── 포스트 목록
│   ├── 포스트 상세
│   └── 댓글
├── 사이드바
│   ├── 프로필 위젯
│   ├── 검색 위젯
│   ├── 카테고리 위젯 (계층적 구조)
│   └── 최근 글 위젯
├── 푸터 (Footer)
└── JavaScript
    └── 계층적 카테고리 생성 스크립트
```

- **기본 폰트**: 나눔명조 (Nanum Myeongjo)
- **숫자 폰트**: Georgia
- **기본 크기**: 14px
- **제목 크기**: 24px (헤더), 18px (포스트)

## 반응형 웹 브레이크포인트

### 🖥️ 데스크톱 (1025px 이상)
- 2단 레이아웃 (메인 콘텐츠 + 사이드바)
- 최대 너비 1200px
- 넓은 여백과 간격

### 💻 태블릿 가로 (769px ~ 1024px)
- 2단 레이아웃 유지
- 좁아진 여백
- 헤더 폰트 크기 조정

### 📱 태블릿 세로 (481px ~ 768px)
- 1단 레이아웃으로 전환
- 사이드바가 콘텐츠 하단으로 이동
- 폰트 크기 및 여백 조정
- 네비게이션 메뉴 자동 줄바꿈

### 📱 모바일 (481px 이하)
- 1단 레이아웃
- 더 작은 폰트와 여백
- 터치 친화적인 버튼 크기
- 최소 여백으로 최대 콘텐츠 표시

### 📱 작은 모바일 (360px 이하)
- 초소형 화면 최적화
- 필수 정보 위주 표시
- 더욱 압축된 레이아웃

## 문제 해결

### 템플릿이 적용되지 않을 때
1. XML 파일이 올바르게 업로드 되었는지 확인
2. 브라우저 캐시를 삭제하고 새로고침 (Ctrl + F5)
3. 백업한 템플릿으로 복원 후 다시 시도

### 계층적 카테고리가 표시되지 않을 때
1. **라벨에 ">" 구분자가 있는지** 확인
   - 구분자가 없으면 카테고리에 표시되지 않습니다
2. 브라우저 콘솔(F12)에서 JavaScript 오류 확인
3. Blogger 블로그가 **비공개 상태**인 경우 JSON Feed 접근이 제한될 수 있습니다
   - 블로그를 공개로 설정하거나
   - 템플릿의 fetch URL을 확인하세요
4. 캐시 문제일 수 있으니 강제 새로고침 (Ctrl + Shift + R)

### 댓글이 표시되지 않을 때
1. Blogger 대시보드 → **설정** → **게시물, 댓글 및 공유**
2. 댓글 허용 여부 확인
3. 댓글 위치를 "포스트 아래"로 설정
4. 포스트에서 댓글 허용 여부 확인

### 프로필 이미지가 표시되지 않을 때
1. Blogger 대시보드 → **레이아웃**
2. 프로필 위젯 설정에서 프로필 사진이 업로드되어 있는지 확인
3. 프로필 위젯이 활성화되어 있는지 확인
